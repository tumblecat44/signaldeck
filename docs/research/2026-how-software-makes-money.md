# 2026에 소프트웨어로 돈 버는 법 — 과한 조사, chill 해석

**조사일:** 2026-07-12  
**방법:** 서브에이전트 6 + 반박 1 + 1차 웹/X 교차  
**자세:** 숫자는 참고용. 자기 신고·생존자 편향·마케팅 문장을 과신하지 않는다.  
**목적:** “핫한 한 방” 찾기보다 **분포와 병목** 파악.

---

## 0. 30초 버전 (진짜로 이 정도만 기억해도 됨)

1. **공개된 인디 SaaS 중앙값은 대략 월 $145–$170.** 평균 $4k는 꼬리가 끌어올린 착시.  
2. **$10k MRR 이상은 양의 매출 표본의 약 6%.** 흔한 일이 아니다.  
3. **만들기 싸짐 → 공급 폭증 → 병목은 유통·신뢰·결제.** levelsio 밋업 관찰과 방향 일치.  
4. **붐비는 카테고리(AI) ≠ 중앙 수익 높은 카테고리.** AI 중앙 ~$150–200, Sales 쪽이 중앙 더 높지만 표본 작음.  
5. **얇은 AI 래퍼·unlimited LTD·인디끼리 팔기는 기대값 나쁨.** 수직 워크플로·결과 과금·배포 자산은 예외 조건 있을 때.  
6. **한국 1인:** 달러 PLG+MoR(Polar) vs 국내 관계형 B2B+로컬 페이는 **다른 게임.**  
7. **이 문서가 “이걸 만들면 돈 된다”고 말하진 않음.** 기댓값 정렬용.

## 1. 데이터 먼저 (영수증에 가까운 것)

### 1.1 TrustMRR / BigIdeasDB 계열 (2026)

| 지표 | 대략 값 | 신뢰 |
|------|---------|------|
| 추적 스타트업 | ~8,700 | med-high |
| 양의 MRR 있는 곳 | ~3,800 | med-high |
| **중앙 MRR** | **~$145** (다른 컷 ~$169) | **high 방향** |
| 평균 MRR | ~$4,300 | 쓰지 말 것 (꼬리) |
| ≥ $1k MRR | ~24% (양의 MRR 분모) | med-high |
| ≥ $10k MRR | ~6% | med-high |
| ≥ $100k MRR | ~0.6% | med-high |
| MoM 성장 중앙 | **0%** (정체/감소 다수) | med |

**해석 (chill):**  
“공개하고 Stripe 연동한 사람들” 안에서도 대부분 작다.  
연동 안 하고 접은 시도까지 넣으면 중앙은 더 0에 가깝다.  
→ **기댓값을 $10k MRR에 두면 정신 건강에 안 좋다.** 현실 1차 목표: $1k–$5k 밴드.

### 1.2 카테고리 (중앙값 기준, 평균 무시)

| 카테고리 | 중앙 MRR 감각 | 메모 |
|----------|---------------|------|
| AI | ~$150–200 | **가장 붐빔**, 대부분 초소형 |
| Productivity | ~$45 | 지옥 경쟁 |
| Dev tools | ~$70 | 마진 좋다는 말 많음, 중앙은 작음 |
| Marketing | ~$275 | 상대적으로 나음 |
| Sales | ~$640 | 중앙 최고 쪽, **표본 작음** 주의 |
| Marketplace | 다른 컷에서 중앙 높음 | 네트워크 효과, 솔로 난이도 다름 |

**한 줄:** AI에 들어가면 “성장률 %”는 예쁠 수 있고(기저 0), **통장 잔고 중앙값은 안 예쁠 수 있다.**

## 2. 2026에 “돈 된다”고 입 모이는 모델 (조건 포함)

순위 아님. **조건부 카탈로그.**

### A. 좁은 B2B / 워크플로 SaaS (self-serve $29–99)
- 한 직업·한 결과물 (이력서, 스레드, 캡션, API 이미지, 설문…)
- 공개 히트 예: Rezi, Typefully, Submagic, Bannerbear 류 패턴
- **조건:** 검색/커뮤니티 의도 트래픽 또는 에이전시 유통

### B. 세일즈·마케팅 인프라 (아웃리치, 어트리뷰션, 리드)
- 중앙 MRR이 AI보다 낫다는 표 쪽
- **조건:** 스팸 규제·플랫폼 ToS·전달성. 도덕/밴 리스크 실재

### C. 플랫폼 애드온 (Shopify, Slack, WP, Notion…)
- 설치 동선이 배포를 대신
- **조건:** 리뷰·승인·수수료. 플랫폼 sherlock 리스크

### D. AI “두꺼운” 제품 (워크플로 락인, 데이터, 컴플라이언스)
- thin wrapper ≠ 전부 사망. Cursor급 컨텍스트는 예외 논의
- 가격: 고정 + usage/크레딧 (unlimited AI 구독은 마진 자살 패턴)
- AI-native GM ~50%대 보고 vs 전통 SaaS 75%+ → **구조적으로 빡셈**

### E. 결과/소비 과금 에이전트 (엔터프라이즈·지원)
- Fin $0.99/resolution, SF Agentforce 가격 피벗 실험
- 인디 복제 난이도 높음. 영수증은 대기업 쪽

### F. 매니지드 AI / 에이전시 리테이너 ($1–5k/mo 주장)
- 존재는 함. X의 “10명 팔면 파이어”는 **코스 판매 극장** 다수
- 서비스 → 점진 제품화는 고전 루트

### G. 정보재 + 어필리에이트 + 비교 콘텐츠
- “카테고리 리스트클 → 어필 커미션” (Steven Tey 등 공개 플레이)
- 소프트웨어 “제품”이 아닐 수 있음. 돈은 됨

### H. 포트폴리오 시리얼 빌더
- levelsio / Marc Lou 식: 여러 작은 것, 죽는 건 죽이고 합산
- **조건:** 배포 채널·브랜드가 이미 있거나, 실험 비용을 감당

### I. 한국 로컬 버티컬
- 법·세무·업종 관행. 글로벌 툴이 못 파는 것
- **조건:** 로컬 페이 + 관계 영업. Polar만으로 안 끝남

## 4. 유통 (Distribution) 2026

### 아직 “된다”는 쪽 (마법 아님)

| 채널 | 언제 | 주의 |
|------|------|------|
| 초개인화 콜드 이메일 | B2B 0→20 | 제네릭 대량 = 죽음 |
| Reddit 도움 먼저 | 의도 트래픽, SEO에도 남음 | 자동화 스팸 밴 |
| SEO | 1→N 복리 | zero-click으로 클릭 줄 수 있음 |
| LinkedIn 콘텐츠+아웃리치 | 비개발 B2B | 시간 소모 |
| 생태계 마켓플레이스 | 애드온 | 승인·경쟁 |
| 입소문/품질 | 전 구간 | 채널이 아니라 결과 |
| X 빌드인퍼블릭 | 파운더 바이어 | 팔로워 0이면 엔진 아님 |

### 과대광고 쪽

- “AEO/GEO만 하면 된다” — 변화는 실재, **메인 KPI로 삼기엔 이름 과함.** C-SEO Bench류 반박(트릭 무효)도 있음.  
- “PH로 인생 역전”  
- “평균 CAC $X” 하나 — 세그먼트 무시하면 무의미  
- 유료 광고만으로 콜드 스케일 — 리텐션 없으면 소각

### CAC 감각

- 신규 CAC 비율 상승 보고 다수 (방향 med-high)  
- 유료 비중 줄고 오가닉/AEO 비중 소폭 상승 주장 (B2B SaaS 마케팅 통계 2026)  
- **솔로 실무:** 초반은 대화(아웃바운드/커뮤니티), 중반은 SEO·케이스, 유료는 측정용

## 6. Lead Radar / “주간 영업 큐” 아이디어 재평가

| 질문 | 답 |
|------|-----|
| 2024 패턴이냐? | **알림 자체는 예.** |
| 2026에 죽냐? | 얇은 키워드 알림은 과밀·저가 경쟁. |
| 살릴 조건? | **니치 ICP + 의도 분류 + 답장 초안 + usage 정렬 가격** = “알림”이 아니라 **주간 파이프라인** |
| 데이터상 위치? | Sales/Marketing 인프라 쪽 — 중앙이 AI보다 나을 **가능성**. 증명 아님. |
| 리스크 | 플랫폼 ToS, 스팸 인식, 인디 고객만 모이면 ARPU 낮음 |
| chill 판정 | **합리적 실험**, “돈 많이 벌겠다” 확언 금지. 2주 배포 → 결제 테스트 → 4주 유료 신호 |

대안 축 (조사 후 후보군, 역시 확언 없음):

1. **Boring B2B 한 업종 워크플로** (스케줄, 정산, 서류) — 영업 필요  
2. **플랫폼 애드온 하나** — 배포 내장  
3. **비교/어필 콘텐츠 사업** — 제품 최소  
4. **GEO/AI visibility** — 하이프 많음, 측정·신뢰가 전부  
5. **크리에이터 단일 산출물 툴** (숏폼 캡션 등) — 틱톡 배포, 수명 짧을 수 있음

## 7. 반박 노트 (과신 방지) — devil’s advocate 에이전트 포함

| 유행 문장 | 반박/뉘앙스 |
|-----------|-------------|
| “빌드 공짜, 배포만 하면 됨” | 배포 없는 빌드는 낭비. 반대로 **쓰레기 제품+배포**도 환불·평판 폭탄. 지원·신뢰·통합 비용은 안 싸짐. |
| “AI 래퍼 전멸” | **얇은** 래퍼 기대값 붕괴. 워크플로·데이터·한국어 도메인 패키징은 별개. Cursor≠Jasper. |
| “중앙 $145니까 관둬” | 중앙은 ‘아무 Stripe나 연동한 군중’ 기댓값. **조건부 노력(니치×실행×개월)** 의 EV와는 다른 질문. |
| “콜드메일 만능” | spray는 더 죽음. 저볼륨·초개인화는 아직 B2B 도구. 한국은 카톡/소개 비중이 더 큼. |
| “무조건 글로벌” | 로컬 해자·네트워크 있으면 국내 합리적. 스케일 천장만 다르게 설계. 이분법 금지. |

## 11. 닫는 말 (Grok chill)

2026에 소프트웨어로 돈 버는 방법은 **없어지지 않았다.**  
사라진 건 “주말에 예쁜 AI 붙여서 수동으로 돈 찍기” 난이도 할인이다.

남는 게임은 지루하다:

> **좁은 고통 × 지불 능력 × 도달 가능한 채널 × 마진 되는 가격 × 출시 반복**

데이터는 “대부분 작다”고 말한다.  
그건 절망이 아니라 **기대값 설정**이다.
